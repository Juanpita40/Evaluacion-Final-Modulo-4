Archivo Readme:

	Proyecto  “Evaluación Final-Módulo 4”.

	Descripción del proyecto: el presente proyecto consiste en la realización de páginas entrelazadas que guardan los datos relativos a un sistema de información 	de una empresa de SSL. Dichas páginas tienen como objetivo formar parte de un sitio empresarial de carácter general, que incluirá todos los componentes del 	presente curso, es decir; tanto el front como el backend necesario para su óptima ejecución.

	Estado del proyecto: finalizada la parte de diseño, base de datos, JAVA EE y subido a los respectivos repositorios de github.
 
	Tecnologías involucradas: hasta ahora se han utilizado HTML5, CSS3, Javascript, Jquery, Diseño Responsivo, Bases de Datos SQL Developer-Oracle, JAVA EE y 	Github.

	Bugs conocidos: ninguno hasta ahora.
